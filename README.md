# A Level Computer Science · OCR H446

A self-contained interactive lesson on **Network Security: Prevention** — built as a single HTML file with no build step, designed for delivery at **The Sixth Form Centre Guernsey** in a 50-minute observed lesson slot.

## Open it

Open `network-security-prevention-jupyter.html` in any modern browser. No npm, no dependencies, no server needed. Only loads Google Fonts; otherwise fully offline.

## Specification mapping

- **§1.3.3 (c)** — Network security and threats; use of firewalls, proxies and encryption *(today's focus)*
- **§1.3.3 (b)** — Protocol layering *(touched via the encapsulation visual)*

Official sources are linked from inside the lesson — the OCR H446 specification PDF and the OCR Delivery Guide for 1.3.3 Networks.

## Lesson structure

| # | Section | Pedagogy |
|---|---|---|
| 0 | Cover + spec reference card | Anchor for the panel |
| 1 | Objectives + recap chips (with pre-lesson video link) | Retrieval practice |
| 2 | Defence in depth (three concentric rings) | Big-picture map |
| 3 | Firewalls + interactive 6-packet simulator | Activity |
| 4 | How a packet is built (encapsulation visual) | Direct instruction |
| 5 | Proxies + with/without-proxy toggle demo | Demo |
| 6 | Caesar cipher — **PRIMM** mini-cycle (Predict → Run → Investigate → Modify) | PRIMM |
| 7 | Asymmetric encryption (Alice/Bob/Eve demo + comparison table) | Direct + demo |
| 8 | Parsons puzzle — order the steps of asymmetric encryption | Active recall |
| 9 | Other defences (AV, backups, passwords, AUP, training, logging) | Direct |
| 10 | Real-world breaches (5 cases, Guernsey-anchored) | Case study |
| 11 | Drag-and-drop threat → defence matcher | Active recall |
| 12 | "Make" — design a defence-in-depth plan for a local bookshop | Application |
| 13 | 6-mark exam-style question with reveal mark scheme | Exam practice |
| 14 | Discussion — Anthropic's Claude Mythos / Project Glasswing | Forward look |
| 15 | 60-second plenary with countdown timer | Plenary |
| 16 | Reflection prompts (exit-ticket style) | Reflection |
| 17 | Download lesson record (single-file HTML, no teacher notes) | Save your work |

## Top-bar toggles

- **⏱ Timings · 50 min** — overlays a per-section minute budget on every card. Toggled by the panel/teacher to see the pacing plan.
- **Teacher notes** — reveals the amber teacher-notes blocks inside every section. Hide during student delivery, show during planning.
